# Managed Apache Cassandra Service

## Summary

- prefix: cassandra

## Privileges

### Alter

- description: Grants permission to alter a keyspace or table
- access level: Write
- resource types

```
{
    "keyspace": {
        "resource_type": "keyspace",
        "required": false,
        "condition_keys": [],
        "dependent_actions": []
    },
    "table": {
        "resource_type": "table",
        "required": false,
        "condition_keys": [],
        "dependent_actions": []
    },
    "": {
        "resource_type": "",
        "required": false,
        "condition_keys": [
            "aws:RequestTag/${TagKey}",
            "aws:TagKeys"
        ],
        "dependent_actions": []
    }
}
```
