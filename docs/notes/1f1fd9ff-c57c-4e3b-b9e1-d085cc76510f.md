# Manage Amazon API Gateway

## Summary

- prefix: apigateway

## Privileges

### DELETE

- description: Used to delete resources
- access level: Write
- resource types

```
{
    "apigateway-general": {
        "resource_type": "apigateway-general",
        "required": true,
        "condition_keys": [],
        "dependent_actions": []
    },
    "": {
        "resource_type": "",
        "required": false,
        "condition_keys": [
            "aws:RequestTag/${TagKey}",
            "aws:TagKeys"

}
```
