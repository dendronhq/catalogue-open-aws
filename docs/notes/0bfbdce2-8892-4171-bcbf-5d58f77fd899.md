# Simple Notification Service

## Summary

- prefix: sns

## Privileges

### AddPermission

- description: Adds a statement to a topic's access control policy, granting access for the specified AWS accounts to the specified actions.
- access level: Permissions management
- resource types

```
{
    "topic": {
        "resource_type": "topic",
        "required": true,
        "condition_keys": [],
        "dependent_actions": []
    }
}
```
