### AttachInstances

- description: Attaches one or more EC2 instances to the specified Auto Scaling group.
- access level: Write
- resource types

```
{
    "autoScalingGroup": {
        "resource_type": "autoScalingGroup",
        "required": true,
        "condition_keys": [
            "autoscaling:ResourceTag/${TagKey}",
            "aws:ResourceTag/${TagKey}"
        ],
        "dependent_actions": []

}
```
