# Budget Service
## Summary
- prefix: budgets
## Privileges
### CreateBudgetAction
- description: Grants permissions to create and define a response that you can configure to execute once your budget has exceeded a specific budget threshold.
- access level: Write
- resource types
```
{
    "budgetAction": {
        "resource_type": "budgetAction",
        "required": true,
        "condition_keys": [],
        "dependent_actions": [
            "iam:PassRole"
        ]
    }
}
```
